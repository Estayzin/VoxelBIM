================================================================================
DESCRIPCIÓN POR CARPETA
================================================================================

ROOT - RAÍZ DEL PROYECTO
  Contiene archivos de configuración global, scripts de inicio y 
  documentación principal del proyecto.

/app
  Aplicación principal que contiene:
  - Integraciones con Autodesk (autodesk.html)
  - Aplicación VoxelBIM (voxelbim.html)
  - Librerías JavaScript (Three.js, Web-IFC, OrbitControls)
  - Assets WASM compilados
  - Visor 3D

/aps-worker
  Worker (Cloudflare Workers) para integración con Autodesk Platform Services.

/scripts
  Utilidades y scripts de automatización en Python y JavaScript para 
  compilación y parching de componentes.

/visor
  Aplicación principal construida con Vite y Node.js
  - Punto de entrada: src/main.js
  - Salida compilada: dist/
  - Incluye librería Web-IFC completa
